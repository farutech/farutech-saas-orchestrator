# Farutech Design System

Welcome to the Farutech Design System documentation. This Storybook contains all the components, hooks, utilities, styles, and more that make up our design system.

## Sections

- **Components**: Reusable UI components organized by category.
- **Hooks**: Custom React hooks for common functionality.
- **Styles**: CSS files, design tokens, and styling utilities.
- **Utils**: Utility functions and helpers.
- **Scripts**: Build and versioning scripts.
- **Fonts**: Typography and font assets.

## Getting Started

To use this design system in your project:

```bash
npm install @farutech/sdk-designsystem
```

Then import components as needed:

```tsx
import { Button, Card } from '@farutech/sdk-designsystem';
```

## Contributing

When adding new components or features, ensure to:

1. Add stories for components
2. Document hooks and utilities
3. Update this documentation
