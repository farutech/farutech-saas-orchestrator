# Fonts

Typography and font assets used in the Farutech Design System.

## Primary Font Family

**Nunito Sans**

A friendly, rounded sans-serif typeface that works well for UI text.

### Weights Available
- Regular (400)
- Italic (400)
- Bold (700)
- Bold Italic (700)

### Usage in CSS
```css
font-family: 'Nunito Sans', sans-serif;
```

### Web Fonts
Fonts are loaded via Google Fonts or included in the build.

In `globals.css`:
```css
@import url('https://fonts.googleapis.com/css2?family=Nunito+Sans:ital,wght@0,400;0,700;1,400;1,700&display=swap');
```

## Font Scale

The design system uses a consistent font scale:

- xs: 0.75rem (12px)
- sm: 0.875rem (14px)
- base: 1rem (16px)
- lg: 1.125rem (18px)
- xl: 1.25rem (20px)
- 2xl: 1.5rem (24px)
- 3xl: 1.875rem (30px)
- 4xl: 2.25rem (36px)

## Font Assets

Font files are included in the build output for offline usage.
